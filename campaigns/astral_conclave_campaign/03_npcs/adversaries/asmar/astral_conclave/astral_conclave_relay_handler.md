# Astral Conclave Relay Handler

## Role
Field command operator responsible for temporary command relays, obedience-device telemetry, and emergency portal extraction.

## Profile
Relay Handlers are calm, technical Conclave agents who speak as if every moral objection is just a failure to understand scale. At Sandglass Relay, the handler wants to escape with logs and crystal keys, not win a stand-up fight.

## Typical Use
- Directing temporary relay camps
- Sending command packets to controlled assets
- Opening emergency portals
- Sacrificing equipment or subordinates to preserve the mission

## Sandglass Relay Tactics
- Begins portal stabilization as soon as intruders are identified.
- Uses Command Static and Signal Snare to slow pursuers.
- Escapes if possible; surrenders only if capture is unavoidable and the party has already secured the portal focus.
